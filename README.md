# NvChad Custom Configs

| Bind         | Action                 |
| ------------ | ---------------------- |
| `<ctrl>n`    | Toggle file tree       |
| `<leader>fw` | Live grep              |
| `<leader>ff` | Find files             |
| `<leader>fb` | Find buffer            |
| `<leader>uu` | Update nvchad          |
| `<leader>h`  | Open terminal          |
| `<ctrl>x`    | Open file horizontally |
| `<ctrl>v`    | Open file vertically   |
